# Ahmad Hamid Nur Rozaq — Maintenance Specialist Portfolio

Repositori ini berisi kode sumber untuk situs web portofolio personal premium milik **Ahmad Hamid Nur Rozaq**, seorang *Maintenance Specialist*. Situs ini dirancang dengan antarmuka modern, interaktif, dan responsif menggunakan teknologi web statis tingkat lanjut.

## ✨ Fitur Utama

*   **Premium Typography & Design:** Menggunakan kombinasi font *Inter*, *JetBrains Mono*, dan *Rajdhani* untuk memberikan kesan teknis sekaligus modern.
*   **Custom Interactive Cursor:** Dilengkapi dengan efek lingkaran pelacak (*cursor ring*) interaktif yang mengikuti pergerakan tetikus pengguna.
*   **Dynamic Navigation Panel:** Bilah navigasi pintar yang berubah gaya secara otomatis saat halaman digulir (*scroll-activated blur effect*).
*   **3D Depth Shadowing:** Penerapan variabel CSS khusus untuk menciptakan efek bayangan berlapis dan kedalaman visual (3D effect).
*   **Ultra-Responsive Layout:** Dioptimalkan secara penuh untuk kenyamanan akses di perangkat seluler maupun desktop.

---

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun murni menggunakan teknologi *frontend* modern tanpa *dependency* eksternal yang berat untuk menjaga performa kecepatan pemuatan halaman:

| Komponen | Teknologi / Integrasi |
| :--- | :--- |
| **Struktur** | HTML5 (Semantik) |
| **Gaya & Efek** | CSS3 (Variabel Kustom, Flexbox, Desain Responsif) |
| **Tipografi** | Google Fonts (Inter, JetBrains Mono, Rajdhani) |
| **Animasi** | JavaScript Kustom & Transisi CSS3 |
